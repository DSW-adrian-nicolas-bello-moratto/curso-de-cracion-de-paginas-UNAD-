proposito del sitio

en esta pagina veremos y aprenderemos juntos el uso de las etiquetas css grid y su utilidad en la programacion y la creacion de tus propias paginas web

¿¿que son las css grid??

las css gri son un tipo de etiqueta semantica mediante la que damos valor a css display que ademas nos va a permitir junto a otras propiedades crar maquetas en miras a nuestras paginas web de una manera mas facil a la usada convencionalmente
